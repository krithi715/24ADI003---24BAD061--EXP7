This project demonstrates customer segmentation using two unsupervised learning techniques: K-Means Clustering and Gaussian Mixture Models (GMM). The dataset used is the Mall Customer Segmentation dataset, which contains customer information such as Annual Income, Spending Score, and Age. The dataset is publicly available on Kaggle: https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python.In Scenario 1, K-Means clustering is applied to group customers into distinct clusters based on similarity, using the Elbow Method to determine the optimal number of clusters and evaluating performance using inertia and silhouette score. In Scenario 2, Gaussian Mixture Models are used to perform probabilistic clustering, allowing soft assignment of data points to clusters, with evaluation metrics such as log-likelihood, AIC, BIC, and silhouette score. The project includes data preprocessing, feature scaling, clustering implementation, and visualization techniques such as scatter plots, elbow curves, contour plots, and comparison graphs. A comparative analysis is also performed to highlight the differences between K-Means (hard clustering with spherical clusters) and GMM (soft clustering with flexible, elliptical clusters), providing insights into customer behavior patterns and segmentation strategies.
